Yor Forger Daggers (From SpyxFamily)

Model Description

Right before winter break of my junior year of high school, I had a free period in my English class. That is 67 whole minutes! So I did the only reasonable thing and made another 3D model. This time, they're based off the daggers(?)/needles(?)/weapon Yor Forger uses in SpyxFamily. And perhaps, also inspired by my sister's Yor Forger cosplay.

This project uses a simple revolve to make the model, along with a torus to spin around the finger.

This was also the first project I printed on a Bambu printer!

3D Printer Settings

This model has a maximum length of 15in. It is cut to fit onto a 220x220x220mm base plate.

To print the pole, print multiple copies of the file "Staff 4" - make as many as you want!

Here were my 3D printer settings:
3D printer: Bambu Lab A1
Slicer: BambuStudio
Filament type: PLA
Nozzle temp: 225 degrees C
Plate temp: 55 degrees C
Layer height: 0.2mm
Supports: On
Support type: tree (auto)
The rest of the settings are the default Bambu Handy settings
