Haku Joudai Keychain (From Toilet-Bound Hanako-kun)


Model Description

As I was still developing my 3D modeling skills with smaller projects, I decided to make a keychain inspired by the small spirits that aid Hanako-kun in the manga, Toilet-Bound Hanako-kun.

This was made using some simple extrusions to give the keychain its different layers

3D Printer Settings

Maximum length: ~4.8in
Maximum width: ~3in
Has a ~.2 inch hole at the top to fit jumper ring for keychain. Single print. Intended to fit on a 220x220x220mm base plate.

I use my general 3D printer settings if I did print this: 3D printer: Creality Ender 3 Pro
Slicer: Ultimaker Cura
Filament type: PLA
Layer Height: .2mm
Wall/Top/Bottom Thickness: .8mm
Infill: 20%
Printing Temp: 200 degrees C
Base Plate Temp: 60 degrees C
